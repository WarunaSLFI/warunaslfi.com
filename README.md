# warunaslfi.com

Personal developer portfolio built with Next.js 15, TypeScript, Tailwind CSS, and Motion.

**Live:** [warunaslfi.com](https://www.warunaslfi.com)

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS 4
- **Animations:** Motion (formerly Framer Motion)
- **Deployment:** Vercel

## Getting Started

```bash
git clone https://github.com/WarunaSLFI/warunaslfi.com.git
cd warunaslfi.com
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
src/
├── app/              # Next.js App Router pages and layouts
├── components/
│   ├── ui/           # Reusable UI primitives
│   ├── sections/     # Page sections (Hero, About, Experience, etc.)
│   ├── layout/       # Header, Footer, Navigation
│   └── shared/       # Shared components
├── lib/              # Utilities, types, constants
├── data/             # Content data (projects, experience, skills)
├── hooks/            # Custom React hooks
└── styles/           # Font configuration
```
